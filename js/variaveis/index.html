variáveis

O que são

Servem para armazenar dados para utilizar no código posteriormente.

Como criar

  • var: Forma antiga.
  • let: Forma moderna, permite mudar o valor.
  • const: Valor constante, não pode ser alterado.

Atribuição

O operador = é usado para atribuir um valor a uma variável.

Características

  • Podem começar sem valor (retornam undefined).
  • Não podem ter nomes duplicados.
  • Devem ter nomes significativos.

Nomes

  • Sem espaços.
  • Usa camelCase geralmente.

Utilidades

  • Organizar o código.
  • Trabalhar com os mesmos valores facilmente.
  • Preservar o estado durante a execução.
