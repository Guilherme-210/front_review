Escopo em JavaScript

Descrição

Este exercício explora o conceito de escopo em JavaScript, mostrando como variáveis podem ser acessadas ou não dependendo do contexto em que foram declaradas, e como o escopo influencia o comportamento de variáveis no código.

Exemplo de Código JavaScript

// Escopo global e escopo local
let pokemon = "Charmander";

function evoluir() {
  pokemon = "Charmeleon"; // Modifica a variável no escopo global
}

console.log(pokemon); // Charmander
evoluir();
console.log(pokemon); // Charmeleon

// Escopo local não acessível fora da função
function criarAnimal() {
  let animal = "Gato";
}
criarAnimal();
console.log(animal); // Erro: animal is not defined

// Diferenças entre var e let
function avaliarNota(nota) {
  if (nota > 60) {
    var aprovado = true; // Var acessível fora do bloco
    let situacao = "Aprovado"; // Let limitado ao bloco
  } else {
    var aprovado = false;
    let situacao = "Reprovado";
  }
  
  console.log(nota);
  console.log(aprovado); // Funciona
  console.log(situacao); // Erro: situacao is not defined
}
avaliarNota(83);
avaliarNota(49);

// Var dentro de função não é acessível fora
function ola() {
  var texto = "Olá, mundo!";
}
console.log(texto); // Erro: texto is not defined

// Comportamento de inicialização de var e let
console.log(nome); // undefined (var é "içada")
console.log(sobrenome); // Erro: Cannot access 'sobrenome' before initialization
var nome = "Isaac";
let sobrenome = "Pontes";
console.log(nome); // Isaac
console.log(sobrenome); // Pontes

Explicação

O código acima demonstra o funcionamento do escopo em JavaScript:

  • Escopo global: Variáveis declaradas fora de funções podem ser acessadas em qualquer parte do código.
  • Escopo local: Variáveis declaradas dentro de funções ou blocos (let/const) não podem ser acessadas fora do seu contexto.
  • O var permite o acesso fora de blocos, mas não fora de funções.
  • As variáveis let e const respeitam o escopo de bloco.

Tema Central: Escopo em JavaScript

O que é o escopo de uma variável?

  • É o contexto no qual a variável foi declarada.
  • Define se a variável pode ou não ser acessada em outras partes do código.

Diferenças entre var, let e const

  • var: Permite que a variável seja acessada fora do bloco onde foi declarada, mas é limitada ao escopo da função.
  • let/const: Variáveis declaradas com let ou const ficam limitadas ao escopo do bloco onde foram definidas.
