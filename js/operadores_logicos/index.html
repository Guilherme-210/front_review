Operadores Lógicos

Descrição

Este exercício aborda os operadores lógicos em JavaScript.

E (AND)

O operador && retorna true somente se ambas as partes forem verdadeiras.

p q p && q
T T T
T F F
F T F
F F F

Exemplos

true && true === true
false && true === false

Ou (OR)

O operador || retorna true se pelo menos uma das partes for verdadeira.

p q p || q
T T T
T F T
F T T
F F F

Exemplos

true || false === true
false || false === false

Não (NOT)

O operador ! inverte o resultado.

p !p
T F
F T

Exemplos

!true === false
!false === true
