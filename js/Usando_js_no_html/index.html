Usando JavaScript no HTML

Pressione a tecla F12 para ver o resultado

Utilizando a Tag <script>

A tag <script> permite adicionar código JavaScript diretamente no arquivo HTML. No entanto, isso pode deixar o arquivo muito grande e desorganizado.

Criando Arquivos JavaScript Separados

  • É a melhor forma de organizar o código JavaScript.
  • Criar arquivos com extensão .js e referenciá-los no HTML usando a tag <script src="caminho_do_arquivo.js">.

Ordem dos Arquivos

  • O HTML carrega os arquivos de cima para baixo.
  • A ordem de carregamento pode afetar o funcionamento do código.
  • Arquivos que dependem de outros devem ser carregados depois.

Múltiplos Arquivos

É possível incluir vários arquivos .js no HTML, e a ordem em que eles são carregados é importante para o funcionamento correto.

Posição da Tag <script> no HTML

A tag <script> pode estar em qualquer lugar no HTML, mas é comum colocá-la no final para garantir que todos os elementos HTML sejam carregados antes do JavaScript.
