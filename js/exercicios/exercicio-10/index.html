Exercício 10: Procurando Palíndromos

Descrição

Este exercício identifica se uma palavra é um palíndromo, ou seja, se lida de trás para frente possui as mesmas letras na mesma ordem.

Exemplo de Código JavaScript

let palavra = prompt("Digite uma palavra:");
let palavraInvertida = palavra.split("").reverse().join("");

if (palavra === palavraInvertida) {
    alert(palavra + " é um palíndromo!");
} else {
    alert(palavra + " não é um palíndromo. Invertida: " + palavraInvertida);
}

Explicação

O código acima pede uma palavra ao usuário e verifica se é um palíndromo.

  • A palavra é lida usando prompt.
  • A palavra invertida é criada usando métodos de manipulação de strings.
  • Uma condição verifica se a palavra original é igual à palavra invertida.
  • O resultado é exibido em um alerta.

Resolução

const palavra = prompt('Escreva uma palavra: ')
let palavraInvertida = ''

for (let i = palavra.length - 1; i >= 0; i--) {
  palavraInvertida += palavra[i];
}


if (palavra === palavraInvertida) {
  alert(`Essa palavra é um palíndromo.\n${palavra}\n${palavraInvertida}`)
} else {
  alert(`Não e um palíndromo.\n${palavra}\n${palavraInvertida}`)
}
