Exercício 4: Cálculo de Dano

Escreva um programa que permita inserir o nome e o poder de ataque de um personagem, depois o nome, a quantidade de pontos de vida, o poder de defesa de outro personagem e se ele possui um escudo. Em seguida, o programa deve calcular a quantidade de dano causado com base nas seguintes regras:

  • Se o poder de ataque for maior do que a defesa e o defensor não possuir um escudo, o dano causado será igual à diferença entre o ataque e a defesa.
  • Se o poder de ataque for maior do que a defesa e o defensor possuir um escudo, o dano causado será igual à metade da diferença entre o ataque e a defesa.
  • Se o poder de ataque for menor ou igual à defesa, o dano causado será 0.

Por fim, o programa deve subtrair a quantidade de dano da quantidade de pontos de vida do personagem defensor e exibir na tela a quantidade de dano e as informações atualizadas de ambos os personagens.

// Exemplo de lógica a ser seguida em JavaScript:

let nomeAtacante = prompt("Digite o nome do personagem atacante:");
let poderAtaque = prompt("Digite o poder de ataque de " + nomeAtacante + ":");

let nomeDefensor = prompt("Digite o nome do personagem defensor:");
let pontosVida = prompt("Digite os pontos de vida de " + nomeDefensor + ":");
let poderDefesa = prompt("Digite o poder de defesa de " + nomeDefensor + ":");
let possuiEscudo = prompt(nomeDefensor + " possui um escudo? (sim/não)").toLowerCase();

let danoCausado = 0;

if (poderAtaque > poderDefesa) {
  if (possuiEscudo === "sim") {
    danoCausado = (poderAtaque - poderDefesa) / 2;
  } else {
    danoCausado = poderAtaque - poderDefesa;
  }
} else {
  danoCausado = 0;
}

pontosVida -= danoCausado;

alert(nomeAtacante + " causou " + danoCausado + " de dano em " + nomeDefensor);
alert(nomeDefensor + " agora tem " + pontosVida + " pontos de vida restantes.");
