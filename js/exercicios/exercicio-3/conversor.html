Exercício 5: Conversor de medidas

Objetivo

Escrever um programa em JavaScript que funcione como um conversor de medidas. O usuário deverá inserir um valor em metros e escolher para qual unidade de medida deseja converter.

As opções disponíveis são:

  • Milímetro (mm)
  • Centímetro (cm)
  • Decímetro (dm)
  • Decâmetro (dam)
  • Hectômetro (hm)
  • Quilômetro (km)

Regras

O programa deverá:

  • Pedir um valor em metros.
  • O usuário escolherá a unidade de medida para a conversão.
  • Converter o valor de acordo com a opção escolhida.
  • Se uma opção inválida for inserida, exibir a mensagem "Opção inválida".

Estrutura de Código

// Pedir um valor em metros
let valorMetros = prompt("Insira o valor em metros:");

// Opções de conversão
let opcao = prompt("Escolha a unidade de conversão:\n1 - milímetros (mm)\n2 - centímetros (cm)\n3 - decímetros (dm)\n4 - decâmetros (dam)\n5 - hectômetros (hm)\n6 - quilômetros (km)");

// Conversão com switch
switch(opcao) {
  case "1":
    alert(`${valorMetros} metros = ${valorMetros * 1000} milímetros`);
    break;
  case "2":
    alert(`${valorMetros} metros = ${valorMetros * 100} centímetros`);
    break;
  case "3":
    alert(`${valorMetros} metros = ${valorMetros * 10} decímetros`);
    break;
  case "4":
    alert(`${valorMetros} metros = ${valorMetros / 10} decâmetros`);
    break;
  case "5":
    alert(`${valorMetros} metros = ${valorMetros / 100} hectômetros`);
    break;
  case "6":
    alert(`${valorMetros} metros = ${valorMetros / 1000} quilômetros`);
    break;
  default:
    alert("Opção inválida!");
}
