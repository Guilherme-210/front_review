Exercício 8: Controle Financeiro

Descrição

Este é um exemplo de um programa de controle financeiro. O usuário pode adicionar ou remover dinheiro e o programa exibirá o saldo atualizado até que o usuário escolha a opção de sair.

Exemplo de Código JavaScript

function mostrarMenu(saldo) {
    return prompt(
        "Saldo atual: R$ " + saldo + "\n" +
        "Escolha uma das opções abaixo:\n" +
        "1. Adicionar dinheiro\n" +
        "2. Remover dinheiro\n" +
        "3. Sair"
    );
}

let saldo = parseFloat(prompt("Digite a quantidade inicial de dinheiro disponível:"));
let opcao = "";

do {
    opcao = mostrarMenu(saldo);

    switch (opcao) {
        case "1":
            let adicionar = parseFloat(prompt("Quanto deseja adicionar?"));
            if (!isNaN(adicionar)) {
                saldo += adicionar;
                alert("Você adicionou R$ " + adicionar + ". Saldo atual: R$ " + saldo);
            } else {
                alert("Valor inválido.");
            }
            break;
        case "2":
            let remover = parseFloat(prompt("Quanto deseja remover?"));
            if (!isNaN(remover)) {
                if (remover <= saldo) {
                    saldo -= remover;
                    alert("Você removeu R$ " + remover + ". Saldo atual: R$ " + saldo);
                } else {
                    alert("Você não pode remover mais do que o saldo disponível.");
                }
            } else {
                alert("Valor inválido.");
            }
            break;
        case "3":
            alert("Saindo do programa...");
            break;
        default:
            alert("Opção inválida! Tente novamente.");
    }
} while (opcao !== "3");

Explicação

O código JavaScript cria um controle financeiro simples que permite ao usuário adicionar ou remover dinheiro e exibir o saldo atualizado.

  • O saldo inicial é inserido pelo usuário no início do programa.
  • O menu interativo permite adicionar, remover dinheiro ou sair do programa.
  • Se o usuário tentar remover mais dinheiro do que o saldo disponível, uma mensagem de erro é exibida.
  • A opção de sair encerra o programa com uma mensagem de despedida.
