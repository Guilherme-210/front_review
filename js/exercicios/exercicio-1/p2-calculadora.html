Exercício 2: Calculadora de 4 Operações.

Descrição

Escreva um programa em JavaScript que permita inserir dois valores numéricos e então calcule o resultado das quatro operações básicas:

  • Soma
  • Subtração
  • Multiplicação
  • Divisão

Após calcular os resultados, o programa deve exibi-los na tela.

Exemplo de Código JavaScript

let valor1 = parseFloat(prompt("Digite o primeiro valor:"));
let valor2 = parseFloat(prompt("Digite o segundo valor:"));

let soma = valor1 + valor2;
let subtracao = valor1 - valor2;
let multiplicacao = valor1 * valor2;
let divisao = valor2 !== 0 ? (valor1 / valor2) : "Erro: Divisão por zero!";

alert(
  "Resultados:\n" +
  "Soma: " + soma + "\n" +
  "Subtração: " + subtracao + "\n" +
  "Multiplicação: " + multiplicacao + "\n" +
  "Divisão: " + divisao
);
