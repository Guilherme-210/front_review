Exercício 1: Cadastro de Recrutas

Descrição

Escreva um programa em JavaScript que permita salvar informações de um recruta. As informações a serem salvas são:

  • O primeiro nome
  • O sobrenome
  • O campo de estudo
  • O ano de nascimento

Depois o programa deve exibir o nome completo do recruta, seu campo de estudo e sua idade (apenas baseada no ano de nascimento).

Exemplo de Código JavaScript

let primeiroNome = prompt("Digite o primeiro nome do recruta:");
let sobrenome = prompt("Digite o sobrenome do recruta:");
let campoDeEstudo = prompt("Digite o campo de estudo do recruta:");
let anoDeNascimento = prompt("Digite o ano de nascimento do recruta:");

let anoAtual = new Date().getFullYear();
let idade = anoAtual - anoDeNascimento;

alert(
  "Nome Completo: " + primeiroNome + " " + sobrenome + "\n" +
  "Campo de Estudo: " + campoDeEstudo + "\n" +
  "Idade: " + idade
);
