Exercício 9: Robô da Tabuada

Descrição

Este exercício pede ao usuário um número de 1 a 20 e calcula a tabuada desse número, exibindo os resultados.

Exemplo de Código JavaScript

let numero = parseInt(prompt("Digite um número de 1 a 20:"));
let resultado = "";

if (numero >= 1 && numero <= 20) {
    for (let i = 1; i <= 20; i++) {
        resultado += "${numero} x ${i} = ${numero * i}\\n";
    }
    alert(resultado);
} else {
    alert("Número inválido! Por favor, digite um número entre 1 e 20.");
}

Explicação

O código acima pede um número ao usuário e calcula a tabuada desse número, exibindo os resultados em um alerta.

  • O número é lido usando prompt.
  • Um loop FOR calcula a tabuada de 1 a 20.
  • Os resultados são armazenados em uma string e exibidos ao final.
  • Uma verificação garante que o número esteja entre 1 e 20.

Resolução

let val = parseFloat(prompt('Insira um valor para ser multiplicado:'))
let valores = [];
let resultado = "";
let separador = " - "; // Defina o separador desejado

for (let i = 0; i <= '20'; i++) {
  resultado = val * i
  valores += `${val} * ${i} = ${resultado}\n`
  // if (i < '20') {
  //   valores += separador
  // }
}

alert(valores);
