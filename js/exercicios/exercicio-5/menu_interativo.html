Menu Interativo

Descrição

Este é um exemplo de um programa que simula um menu interativo com 5 opções diferentes. A última opção encerra o programa.

Exemplo de Código JavaScript

function mostrarMenu() {
    return prompt(
        "Escolha uma das opções abaixo:\n" +
        "1. Opção 1\n" +
        "2. Opção 2\n" +
        "3. Opção 3\n" +
        "4. Opção 4\n" +
        "5. Encerrar"
    );
}

let opcao = "";

do {
    opcao = mostrarMenu();

    switch (opcao) {
        case "1":
            alert("Você escolheu a Opção 1");
            break;
        case "2":
            alert("Você escolheu a Opção 2");
            break;
        case "3":
            alert("Você escolheu a Opção 3");
            break;
        case "4":
            alert("Você escolheu a Opção 4");
            break;
        case "5":
            alert("Encerrando o sistema...");
            break;
        default:
            alert("Opção inválida! Tente novamente.");
    }
} while (opcao !== "5");

Explicação

O código JavaScript cria um menu interativo que permite ao usuário escolher entre 5 opções. O loop do while continua exibindo o menu até que o usuário escolha a opção de encerrar (opção 5).

  • As opções 1 a 4 apenas mostram mensagens indicando qual opção foi selecionada.
  • A opção 5 exibe uma mensagem de encerramento e finaliza o loop.
  • Se o usuário inserir uma opção inválida, uma mensagem de erro é exibida.
