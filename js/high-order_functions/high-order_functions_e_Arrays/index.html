High-Order Functions e Arrays

Descrição

Nesta aula, vamos aprender sobre 4 métodos importantes para trabalhar com arrays em JavaScript: map, filter, reduce e sort.

Exemplo de Código JavaScript

const personagens = [
  { nivel: 42, nome: "Thrall", raca: "Orc", classe: "Xamã" },
  { nivel: 28, nome: "Garrosh", raca: "Orc", classe: "Guerreiro" },
  { nivel: 35, nome: "Varok", raca: "Orc", classe: "Guerreiro" },
  { nivel: 35, nome: "Uther", raca: "Humano", classe: "Paladino" },
  { nivel: 26, nome: "Jaina", raca: "Humano", classe: "Maga" },
  { nivel: 39, nome: "Tyrande", raca: "Elfo Noturno", classe: "Sacerdotisa" },
  { nivel: 29, nome: "Muradin", raca: "Anão", classe: "Guerreiro" }
];


// map: Cria um novo array com os nomes dos personagens
const nomes = personagens.map(function (personagem) {
  return personagem.nome;
});


// filter: Cria um novo array apenas com os personagens da raça "Orc"
const orcs = personagens.filter(function (personagem) {
  return personagem.raca === "Orc";
});


// reduce: Calcula o nível total dos personagens
const nivelTotal = personagens.reduce(function (acumulador, personagem) {
  return acumulador + personagem.nivel;
}, 0);


// sort: Ordena os personagens pelo nível
personagens.sort(function (a, b) {
  return a.nivel - b.nivel;
});


console.log(nomes);
console.log(orcs);
console.log(nivelTotal);
console.log(personagens);

Explicação

  • map: Retorna um novo array com o resultado de uma função aplicada a cada elemento do array original.
  • filter: Retorna um novo array com elementos que passaram no teste fornecido por uma função.
  • reduce: Acumula os valores de um array em um único resultado final, conforme definido por uma função.
  • sort: Ordena os elementos do array. Modifica o array original.
