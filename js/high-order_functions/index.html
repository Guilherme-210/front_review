High-Order Functions

Descrição

Nesta aula, vamos explorar o conceito de High Order Functions (Funções de Alta Ordem) em JavaScript. Funções de Alta Ordem são funções que recebem outras funções como parâmetro e as executam dentro delas.

Elas são uma parte essencial do paradigma funcional e permitem criar código mais modular, flexível e reutilizável.

Exemplo de Código JavaScript

function calcular(a, b, operacao) {
  console.log("Realizando uma operação.")
  const resultado = operacao(a, b)
  return resultado
}

function somar(x, y) {
  console.log("Realizando soma.")
  return x + y
}

calcular(3, 5, somar)

calcular(9, 2, function (x, y) {
  console.log("Realizando subtração.")
  return x - y
})

const lista = ["Maçã", "Banana", "Laranja", "Limão", "Uva"]

lista.forEach(function (elemento, indice) {
  console.log("Elemento:", elemento, "- Índice:", indice)
})

Explicação

  • A função calcular é uma High Order Function, pois recebe a função operacao como parâmetro e a executa.
  • Usamos a função somar como um exemplo de callback, e também passamos uma função anônima para realizar a subtração.
  • O método forEach percorre o array lista, executando uma função callback para cada elemento.
