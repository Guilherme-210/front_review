Exercício 16: Escalação do Time Home Trabalhando com eventos no Javascript

Enviando valores do DOM para o Javascript

Descrição do exercício

Interagir com o DOM passando valores para dentro das funções Javascript

  • Utilizar o parâmetro this para referenciar o próprio elemento que dispara o evento.
  • Acessar elementos pai e filhos para capturar valores.

Execução do exercício




Resolução do código


function register(section) {
  const username = section.children[1].value;
  const password = section.children[3].value;
  const passwordConfirmation = section.children[5].value;

  if (password === passwordConfirmation) {
    alert(`Usuário ${username} registrado com sucesso!`);
  } else {
    alert('As senhas não conferem!');
  }
}
