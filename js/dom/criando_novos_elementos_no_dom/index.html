Criando Novos Elementos no DOM

Descrição

No JavaScript, podemos criar dinamicamente novos elementos na página HTML usando o DOM. Isso permite adicionar conteúdo e funcionalidades sem a necessidade de recarregar a página.

Os passos para criar e adicionar novos elementos são:

  • Criar o novo elemento usando document.createElement().
  • Adicionar conteúdo ou atributos ao novo elemento.
  • Anexar o novo elemento a um elemento pai existente usando appendChild().

Exemplo de Código JavaScript

function addInput() {
  const ul = document.getElementById("inputs")

  // Criar um novo elemento li
  const newLi = document.createElement("li")
  newLi.className = "list-item"
  newLi.innerText = "Novo input: "

  // Criar um novo elemento 
  const newInput = document.createElement("input")
  newInput.type = "text"
  newInput.name = "input"

  // Adicionar o input ao li
  newLi.appendChild(newInput);

  // Adicionar o li à lista ul
  ul.appendChild(newLi)
}

    Explicação

    • createElement: Cria um novo elemento HTML. No exemplo, usamos para criar um <li> e um <input>.
    • appendChild: Adiciona um elemento como filho de um elemento pai existente. O <li> é adicionado à <ul>.
