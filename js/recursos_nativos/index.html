Recursos nativos do navegador

Console

  • console.log() - mostra mensagens no console.
  • console.error() - mostra mensagens de erro no console.
  • console.warn() - mostra avisos no console.

Window

  • window.alert() - mostra alertas/popups.
  • window.prompt() - permite entrada de dados pelo usuário.
  • window.confirm() - confirmação booleana (sim/não).
  • É o objeto global, então seus métodos podem ser chamados sem o window..

Document

Representa o documento HTML. Útil para interagir com a página mais tarde.
