Tipos de dados

Descrição

Este exercício explora os principais tipos de dados em JavaScript, tanto primitivos quanto complexos, essenciais para manipulação de informações.

Tipos Primitivos

  • Number: Inclui números inteiros e decimais (floats).
  • BigInt: Para representar números inteiros muito grandes.
  • String: Texto delimitado por aspas, pode ser acessado como um array de caracteres.
  • Boolean: Tipo lógico com dois valores: true ou false.
  • Null: Representa um valor nulo, ou "vazio".
  • Undefined: Valor não definido, para variáveis sem valor atribuído.

Outros Tipos Importantes

  • Objeto: Estrutura que agrupa múltiplos valores, como o objeto console.
  • Função: Blocos de código que realizam tarefas específicas, como o console.error().

Pontos de Estudo

  • Diferença entre números inteiros e floats.
  • Como usar BigInt para representar números grandes.
  • Regras de uso para Strings (aspas).
  • Definição e uso de valores booleanos.
  • Diferença entre Null e Undefined.
  • Uso de objetos como estruturas complexas.
  • Funções como métodos que realizam operações no código.
