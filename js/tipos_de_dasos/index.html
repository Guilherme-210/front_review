Tipos de dados

Tipos primitivos

  1. Number: Tipo numérico tradicional, inclui inteiros e números com vírgula (floats)
  2. BigInt: Tipo usado para representar números inteiros muito grandes
  3. String: Tipo textual, deve estar entre aspas. Pode ser acessado como uma lista de caracteres
  4. Boolean: Tipo lógico, possui apenas dois valores: true ou false
  5. Null: Valor nulo, representa um valor vazio
  6. Undefined: Valor não definido, quando algo não teve um valor atribuído

Outros tipos importantes

  • Objeto: Estrutura que armazena múltiplos valores, como o próprio console
  • Função: Trechos de código que executam tarefas, como o console.error
