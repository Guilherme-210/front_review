Novas Formas de Trabalhar com Arrays

Descrição

Esta aula aborda novas formas de manipular arrays no JavaScript, permitindo uma maior flexibilidade e dinamismo.

Adicionar Elementos

let frutas = ["maçã", "banana"];
frutas.push("laranja"); // adiciona "laranja" no final
frutas.unshift("morango"); // adiciona "morango" no começo
console.log(frutas); // ["morango", "maçã", "banana", "laranja"]

Remover Elementos

frutas.pop(); // remove "laranja" do final
frutas.shift(); // remove "morango" do começo
console.log(frutas); // ["maçã", "banana"]

Pesquisar por um Elemento

console.log(frutas.includes("banana")); // true
console.log(frutas.indexOf("maçã")); // 0

Cortar e Concatenar Arrays

let frutasCortadas = frutas.slice(0, 1); // cria uma cópia da parte do array
console.log(frutasCortadas); // ["maçã"]

O método slice() não modifica o array original, apenas retorna uma cópia.

Adicionando Elementos

let frutas = ["maçã", "banana"];
frutas.push("laranja"); // adiciona no final
frutas.unshift("morango"); // adiciona no início
console.log(frutas); // ["morango", "maçã", "banana", "laranja"]

Removendo Elementos

frutas.pop(); // remove do final
frutas.shift(); // remove do início
console.log(frutas); // ["maçã", "banana"]

Pesquisando Elementos

console.log(frutas.indexOf("banana")); // 1
console.log(frutas.includes("maçã")); // true

Cortando e Concatenando

let frutasCortadas = frutas.slice(0, 1); // ["maçã"]
let novasFrutas = frutas.concat(["kiwi", "abacaxi"]); // ["maçã", "banana", "kiwi", "abacaxi"]

O método slice() retorna uma parte do array existente, enquanto concat() concatena arrays e valores.

Substituindo Elementos

frutas.splice(1, 1, "manga"); // remove 1 elemento no índice 1 e adiciona "manga"
console.log(frutas); // ["maçã", "manga"]

Iterando Sobre Elementos

for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}

Utilizando um loop for para percorrer o array.

Conclusão

Essas operações permitem trabalhar com arrays de forma dinâmica e flexível em seus programas.
