Arrays Bidimensionais (Matrizes)

Descrição

Este exercício apresenta como trabalhar com arrays bidimensionais (matrizes) no JavaScript, permitindo manipular dados de forma estruturada.

Exemplo de Código JavaScript

let matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

// Acessando elementos
console.log(matriz[1][2]); // Saída: 6

// Iterando sobre a matriz
for (let i = 0; i < matriz.length; i++) {
    for (let j = 0; j < matriz[i].length; j++) {
        console.log("Elemento na posição [" + i + "][" + j + "]: " + matriz[i][j]);
    }
}

// Adicionando uma nova linha
matriz.push([10, 11, 12]);
console.log(matriz);

Explicação

O programa demonstra como criar e manipular uma matriz no JavaScript:

  • Uma matriz é definida como um array de arrays.
  • Os elementos são acessados usando colchetes encadeados, como matriz[1][2].
  • Loops encadeados permitem iterar sobre cada elemento da matriz.
  • O método push() é utilizado para adicionar uma nova linha à matriz.
