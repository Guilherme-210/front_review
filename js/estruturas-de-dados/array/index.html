Arrays no JavaScript

Descrição

Esta aula aborda o conceito de Arrays em JavaScript, que são estruturas de dados para trabalhar com listas.

O que são Arrays

  • Estrutura de dados para trabalhar com listas.
  • Elementos identificados por índices inteiros (começando do 0).
  • Armazena qualquer tipo de dado.
  • Tamanho dinâmico.
  • Várias funções úteis.

Criando Arrays

let frutas = ["maçã", "banana", "laranja"];

Manipulando Arrays

// Acessar posição
console.log(frutas[1]); // banana

// Atribuir valor
frutas[1] = "manga";
console.log(frutas); // ["maçã", "manga", "laranja"]

// Referenciar índice que não existe
console.log(frutas[3]); // undefined

Funções Úteis

Algumas operações comuns com arrays incluem:

  • Enfileirar e desenfileirar elementos (fila).
  • Empilhar e desempilhar elementos (pilha).
  • Descobrir índice de um valor.
  • Cortar e concatenar arrays.
