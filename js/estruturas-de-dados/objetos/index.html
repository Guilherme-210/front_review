Objetos no JavaScript

Descrição

Este exercício apresenta como trabalhar com objetos no JavaScript, permitindo armazenar dados em pares chave-valor.

Exemplo de Código JavaScript

let pessoa = {
    nome: "João",
    idade: 25,
    endereço: {
        rua: "Av. Brasil",
        cidade: "São Paulo"
    },
    apresentar: function() {
        return "Meu nome é " + this.nome + " e tenho " + this.idade + " anos.";
    }
};

// Acessando propriedades
console.log(pessoa.nome); // Saída: João
console.log(pessoa["idade"]); // Saída: 25
console.log(pessoa.endereço.rua); // Saída: Av. Brasil

// Chamando o método
console.log(pessoa.apresentar()); // Saída: Meu nome é João e tenho 25 anos.

Explicação

O programa demonstra como criar e manipular um objeto no JavaScript:

  • Um objeto é definido usando chaves {}, contendo pares chave-valor.
  • Propriedades são acessadas usando o operador ponto ou colchetes.
  • Métodos podem ser definidos como funções dentro de um objeto.
  • Objetos permitem a criação de estruturas de dados complexas e flexíveis.
