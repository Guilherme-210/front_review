Manipulação de atributos com JavaScript

Tema central

  • Como manipular atributos de elementos HTML utilizando JavaScript

Tópicos abordados

Selecionando elementos

  • Utilizando document.getElementById() para selecionar elementos pelo ID

Lendo e alterando atributos

  • Lendo atributos com .getAttribute()
  • Alterando atributos com .setAttribute()
  • Lendo e alterando valores diretamente, como .value

Diferenças importantes

  • .value lê o valor atual, enquanto .getAttribute() lê o valor no HTML
  • Atributos no HTML usam kebab-case, enquanto no JS usa-se camelCase

Atributos customizados data-*

  • Armazenar dados nos elementos com data-
  • Acessar esses dados com .dataset
  • Útil para guardar informações nos elementos

Exemplos na prática

  • Alterando o valor com .value
  • Alternando entre text e radio com .type e .setAttribute()
  • Habilitando/desabilitando com .disabled
  • Definindo placeholder com .placeholder
  • Lendo e alterando data-something

Manipulando Atributos com Javascript

Resolução do código script.js


// Vamos começar pelo valor do input:
const input = document.getElementById('input')

document.getElementById('value').addEventListener('click', function () {
  // input.value = 'Olá, mundo!'
  input.value = input.value === '' ? 'Olá, mundo!' : ''

  console.log(input.value)
  console.log(input.getAttribute('value'))
})

// Também podemos alterar o próprio tipo de input, e isso reflete imediatamente na página:
document.getElementById('type').addEventListener('click', function () {
  input.type = input.type !== 'date' ? 'date' : 'text'
  // input.setAttribute('type', 'radio')
})

// Podemos manipular atributos comuns, como o placeholder:
document.getElementById('placeholder').addEventListener('click', function () {
  input.placeholder = 'Digita algo...'
})

// E também atributos booleanos, como o disabled:
document.getElementById('disable').addEventListener('click', function () {
  input.setAttribute('disabled', !input.disabled)
})

// Por fim, podemos usar o poder dos atributos de dados do HTML dentro do javascript para aumentar ainda mais as nossas possiblidades:
document.getElementById('data').addEventListener('click', function () {
  const data = input.dataset.somethingElse
  console.log('O valor do atributo data-something-else é: ' + data)
  input.dataset.somethingElse = 'Algum outro valor'
  console.log(
    'O valor do atributo data-something-else agora é: ' +
      input.dataset.somethingElse
  )
})
// Obs.: Repare no “Inspecionar” que podemos inserir atributos dinamicamente no elemento.
