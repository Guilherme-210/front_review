Eventos no HTML

Descrição

Eventos no HTML permitem interatividade e ações dinâmicas em uma página web. Ao acionar eventos, é possível executar códigos JavaScript sem travar o carregamento da página, criando uma experiência mais fluida.

Os principais eventos são:

  • onMouseEnter: ao passar o mouse sobre um elemento
  • onChange: ao mudar o valor de um elemento
  • onFocus: ao colocar foco em um elemento
  • onBlur: ao tirar o foco de um elemento
  • onClick: ao clicar em um elemento

Exemplo de Código JavaScript

function mouseEnterEvent() {
  console.log("mouse entered")
}

function changeEvent() {
  console.log("value changed")
}

function focusEvent() {
  console.log("gained focus")
}

function blurEvent() {
  console.log("lost focus")
}

function clickEvent() {
  alert("clicked")
}

Explicação

  • O evento onMouseEnter é disparado quando o mouse entra no botão.
  • O evento onClick é acionado ao clicar no botão.
  • O evento onFocus ocorre ao colocar foco no campo de texto.
  • O evento onBlur é acionado ao tirar o foco do campo de texto.
  • O evento onChange é acionado ao alterar o valor do campo.
