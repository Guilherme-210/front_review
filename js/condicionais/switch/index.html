Estruturas Condicionais: Switch

Descrição

A estrutura condicional switch testa uma expressão/valor contra vários casos possíveis, executando o bloco de código correspondente ao primeiro caso que combina.

É útil quando há várias opções possíveis que podem ser testadas, ao invés de usar múltiplas condições com if.

Vantagens

  • Testa múltiplos casos de forma mais simples que vários ifs encadeados.
  • Permite definir um comportamento padrão com o default.
  • Permite interromper a execução de outros casos com o break.

Exemplo de Estrutura Básica

switch (expressao) {
  case "valor1":
    // código a ser executado
    break;
  case "valor2":
    // código a ser executado
    break;
  default:
    // código a ser executado se nenhum caso for correspondido
}

Funcionamento

A estrutura switch:

  • Avalia a expressão e testa contra os casos (case).
  • Executa o código do primeiro caso correspondente.
  • Se não houver break, continuará executando os próximos casos.
  • O bloco default é executado quando nenhum caso corresponde à expressão.
