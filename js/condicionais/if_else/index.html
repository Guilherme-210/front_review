Estruturas Condicionais: IF e ELSE

Descrição

As estruturas condicionais permitem criar blocos de código que serão executados somente quando uma determinada operação lógica for verdadeira.

Sintaxe IF

if (condicao) {
  // código a ser executado
}

Sintaxe IF ELSE

if (condicao) {
  // código se a condição for verdadeira
} else {
  // código se a condição for falsa
}

Sintaxe IF ELSE IF

if (condicao1) {
  // código se condicao1 for verdadeira
} else if (condicao2) {
  // código se condicao1 for falsa e condicao2 for verdadeira
} else {
  // código se todas as condições forem falsas
}

Operador Ternário

Forma abreviada e autoavaliada do IF ELSE:

const resultado = condicao ? resultado_verdadeiro : resultado_falso;
