Parâmetros de uma função

Descrição

Este exercício explora os parâmetros em funções JavaScript, mostrando como usá-los, definir valores padrão e como organizar múltiplos parâmetros.

Exemplo de Código JavaScript

// O parâmetro é a entrada da função
// Ele é como uma variável que fica disponível dentro da função
// mas que é informada na hora de chamar a função

function dobro(x) {
  console.log("O dobro de " + x + " é " + (x * 2));
}
dobro(5);
dobro(7);

// Parâmetro não informado tem valor undefined
dobro();

// Valor padrão para os parâmetros
function dizerOla(nome = "mundo") {
  console.log("Olá, " + nome + "!");
}
dizerOla("Isaac");
dizerOla();

// Vários parâmetros
function soma(a, b) {
  console.log(a + b);
}
soma(1, 1);
soma(34, 5);

// A ordem dos parâmetros é importante
function criarUsuario(nome, email, senha, tipo = "leitor") {
  const usuario = { nome, email, senha, tipo };
  console.log(usuario);
}

function novoUsuario(nome, tipo = "leitor", email, senha) {
  const usuario = { nome, email, senha, tipo };
  console.log(usuario);
}

criarUsuario("Isaac", "isaac@email.com", "1234");
novoUsuario("Isaac", "isaac@email.com", "1234");

// Funções com muitos parâmetros
function parametrosDoJeitoErrado(nome, telefone, endereco, aniversario, email, senha) {
  // ...
}

function parametrosDoJeitoCerto(usuario) {
  // ...
}

const dadosDoUsuario = {
  nome: "",
  telefone: "",
  email: "",
  senha: "",
  aniversario: "",
  endereco: ""
};
parametrosDoJeitoCerto(dadosDoUsuario);

Explicação

Parâmetros em funções permitem passar dados para dentro das funções no momento em que elas são chamadas. Eles podem ter valores padrão e a ordem dos parâmetros é importante, especialmente quando se define vários parâmetros.

  • Funções podem ter um ou mais parâmetros.
  • Parâmetros não informados ficam com o valor undefined por padrão.
  • Podemos atribuir valores padrão para parâmetros não informados.
  • Em funções com muitos parâmetros, usar um objeto facilita a organização.

Explicação Adicional: Parâmetros de Funções

Tema central: Parâmetros de funções

  • O que são parâmetros: Entradas de uma função, variáveis disponíveis dentro da função, informadas quando a função é chamada.
  • Declarando parâmetros: São declarados dentro dos parênteses na função. Pode ter quantos parâmetros quiser, separados por vírgula.
  • Chamando a função com parâmetros: Os valores são passados dentro dos parênteses, na mesma ordem em que foram declarados. A ordem é muito importante!
  • Parâmetros opcionais: Podem ter valores padrão, e devem ser declarados por último!
  • Dica: Ao usar muitos parâmetros, transforme-os em um objeto. Isso facilita a chamada da função e torna o código mais legível.
