Retorno de Funções no JavaScript

Descrição

Este exercício exemplifica como o retorno de uma função funciona no JavaScript, incluindo o uso de return, suas regras e implicações.

Exemplo de Código JavaScript

function calcularMedia(a, b) {
    const media = (a + b) / 2;
    return media;
}

const resultado = calcularMedia(5, 9);
console.log("A média é:", resultado);

function criarProduto(nome, preco) {
    const produto = { nome, preco, estoque: 1 };
    return produto;
}

const notebook = criarProduto("Notebook Intel Core i3 8GB", 3000);
console.log(notebook);

function areaRetangular(base, altura) {
    return base * altura;
}

console.log(areaRetangular(3, 5));

function areaQuadrada(lado) {
    return areaRetangular(lado, lado);
}

console.log(areaQuadrada(8));

function maioridade(idade) {
    if (idade >= 18) {
        return "Maior de idade";
    } else {
        return "Menor de idade";
    }
}

console.log(maioridade(20));
console.log(maioridade(13));

Explicação

O código mostra como funções retornam valores em diferentes cenários:

  • Funções sem retorno explícito retornam undefined.
  • O return define a saída da função.
  • O valor retornado pode ser armazenado em uma variável ou usado diretamente em outras funções ou expressões.
  • Funções só podem ter um retorno, mas diferentes blocos condicionais (if/else) podem ter retornos diferentes.

Tema Central: Retorno de Funções no JavaScript

O que é o retorno de uma função (return)

  • É o valor de saída que a função devolve.
  • Permite trabalhar com valores gerados dentro da função em outros locais do código.

Utilizando return

  • Atribuindo a uma variável para usar depois: resultado = calcularMedia(5, 7)
  • Chamando a função diretamente: console.log(calcularMedia(5, 7))
  • Pode retornar qualquer expressão, não precisa ser uma variável.
  • Pode chamar outra função: return areaRetangular(lado, lado)

Regras do return

  • Função só pode ter um único return.
  • Depois que é chamado, a função é finalizada.
  • Exceção: Blocos condicionais (if/else) podem ter vários retornos. Cada ramificação pode ter seu próprio return.
