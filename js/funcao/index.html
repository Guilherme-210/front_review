Funções no JavaScript

O que é uma função?

Uma função é uma estrutura que permite declarar um procedimento e executá-lo programaticamente. Trata-se de um bloco de código que pode ser armazenado e executado quando necessário, possibilitando a reutilização do mesmo código várias vezes.

Partes de uma função

  • Entrada: parâmetros/argumentos [opcional]
  • Procedimento: bloco de código a ser executado
  • Saída: retorno [opcional]

Sintaxe

function nomeDaFuncao() {
  // procedimentos
}

function: palavra reservada

nomeDaFuncao: nome escolhido para a função

(): parâmetros (mesmo que vazio)

{}: corpo da função (bloco de código)

Utilização

A função é declarada utilizando a palavra-chave function. Ela é chamada pelo nome, seguido de parênteses:

nomeDaFuncao();

Você pode chamar uma função quantas vezes for necessário.

Resumo

  • Permite reutilizar código.
  • Declara um procedimento.
  • Executa programaticamente com ().
  • Pode ter entradas e saídas (opcionais).
