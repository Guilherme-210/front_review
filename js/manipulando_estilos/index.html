Manipulando estilos com JavaScript

Introdução

  • Até agora trabalhamos com comportamentos das páginas (eventos, formulários, etc)
  • Também podemos manipular os estilos e CSS com JavaScript
  • Vamos ver como implementar temas claro e escuro alternando estilos

Formas de manipular estilos

Editando propriedade style

  • Elementos possuem propriedade style para acessar e manipular estilos inline
  • Propriedades CSS são mapeadas para camelCase (ex: background-color vira backgroundColor)

Alternando classes

  • Podemos acessar lista de classes com element.classList
  • Método toggle adiciona classe se não tiver e remove se tiver
  • Útil quando usamos frameworks CSS como Bootstrap

Implementação

  • Função useLightTheme define cores para tema claro
  • Função useDarkTheme inverte cores para escuro
  • Função switchTheme alterna classes isLight e isDark
  • Adição de eventos de clique nos botões para chamar funções

Observação

  • Mistura de estilos inline e classes pode causar bugs
  • Melhor optar por uma abordagem só

Resolução do código


// Seleciona a primeira div com a classe 'section is-light'
const themeDiv = document.querySelector(".section-dark")

// Manipulando estilos diretamente pela propriedade style
function useLightTheme() {
  themeDiv.style.color = "#000000"
  themeDiv.style.backgroundColor = "#f1f5f9"
}

function useDarkTheme() {
  themeDiv.style.color = "#f4f4f4"
  themeDiv.style.backgroundColor = "#2e2e2e"
}

// Manipulando estilos através das classes utilizadas pelo CSS
function switchTheme() {
  themeDiv.classList.toggle("section-light")
  themeDiv.classList.toggle("section-dark")
}

// Adicionando os eventos
document.getElementById("lightBtn").addEventListener("click", useLightTheme)
document.getElementById("darkBtn").addEventListener("click", useDarkTheme)
document.getElementById("switchBtn").addEventListener("click", switchTheme)
