Estruturas de Repetição: Do While

Conceito

A estrutura de repetição do while garante a execução de um bloco de código pelo menos uma vez, mesmo que a condição seja falsa na primeira verificação.

Sintaxe

do {
    // código a ser executado
} while (condição);

Exemplo

let velocidade = 0;
do {
    alert("Velocidade: " + velocidade);
    velocidade -= 20;
} while (velocidade > 0);
alert("Velocidade final: " + velocidade); // -20

Diferenças para While

  • While: Verifica a condição antes de executar o código.
  • Do While: Executa o código antes de verificar a condição.

Utilidade

Utilizado quando é necessário garantir que o código dentro do laço seja executado pelo menos uma vez, independentemente da condição inicial.
