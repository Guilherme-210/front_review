Estruturas de Repetição: While

O que é?

A estrutura while no JavaScript é uma estrutura de repetição que executa um bloco de código enquanto uma condição específica for verdadeira.

Funcionamento

O laço while é executado repetidamente enquanto a condição for verdadeira. Se a condição nunca se tornar falsa, o loop continuará indefinidamente, causando o que chamamos de "loop infinito".

Sintaxe

while (condição) {
    // código a ser repetido
}

Características

  • Aceita o uso de break para interromper a repetição.
  • Pode ser combinado com outras estruturas como if e switch.

Utilizações

Usado quando precisamos repetir um trecho de código por tempo indeterminado, em programas que precisam ficar executando até que o usuário decida sair, ou dentro de outras estruturas para criar repetições mais complexas.

Exemplo

// Exemplo de loop while
let contador = 0;
while (contador < 5) {
    console.log("O contador é: " + contador);
    contador++; // Incrementa o contador para evitar loop infinito
}
