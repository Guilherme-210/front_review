Estrutura de Repetição FOR

Descrição

Este exercício aborda a estrutura de repetição FOR no JavaScript, explicando suas partes e usos comuns.

Exemplo de Código JavaScript

for (let i = 0; i <= 10; i++) {
    console.log(i); // Contar de 0 a 10
}

Explicação

O código acima usa um loop FOR para contar de 0 a 10 e imprimir cada número no console.

  • Inicialização: let i = 0 define o início da contagem.
  • Condição: i <= 10 garante que a contagem vá até 10.
  • Finalização: i++ incrementa o índice a cada iteração.
